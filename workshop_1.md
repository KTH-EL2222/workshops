# Workshop 1

## Dictionary
* RPi = Raspberry Pi
* PWM = Pulse-width modulation

## Objective

In this workshop you will be familiarized with the hardware and techniques that are going to be used to control robots motion. You will learn how to power up motors and read encoder signal within an Arduino.

For this lab you will need following equipment:
* [Arduino UNO](https://www.arduino.cc/en/Main/ArduinoBoardUno)
* AB USB cable for connecting Arduino to a computer.
* Breadboard
* Jumper wires
* Power supply
* LED and resistor
* L293D (H-bridge)
* Voltage regulator
* Soldering station and soldering iron
* Computer with installed [Arduino IDE](https://www.arduino.cc/en/Main/Software)
